# Examples

Runnable examples and a usage guide for designing, training, and evaluating ChessBot models.

- **`tutorial_usage_and_tips.md`** – Usage guide covering training, evaluation, and the web app.
- **`run_selfplay.py`** – Run a self-play game from the command line.
- **`run_model_comparison.py`** – Run a match between two models.
- **`run_evaluation.py`** – Evaluate a model on the test dataset.
